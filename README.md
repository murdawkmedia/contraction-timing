# Contraction Timer & Box Breathing App

A simple, beautifully designed, and distraction-free contraction timer with an integrated 4-box breathing visualizer. Designed to work well on both mobile and desktop.

## Features

- **Smooth Timer:** High fidelity stop-watch using `requestAnimationFrame`.
- **Contraction History:** Logs duration and interval times.
- **5-1-1 Rule Alert:** Clearly displays when it's time to head to the hospital.
- **Box Breathing:** 4-box breathing method modal (4s inhale, 4s hold, 4s exhale, 4s hold) to calm the nervous system.
- **Privacy-first:** Local browser storage, no accounts required.
